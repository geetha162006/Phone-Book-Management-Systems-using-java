# Phone-Book-Management-Systems-using-java

## Overview
The Phone Book Management System is a Java-based desktop application developed to manage contact information efficiently. The application allows users to add, search, edit, delete, and display contacts through a simple graphical user interface built using Java Swing and AWT.

This project demonstrates the implementation of Object-Oriented Programming concepts, GUI development, and data structure usage in Java.

---

## Features
- Add new contacts
- Search contacts quickly
- Edit existing contact details
- Delete contacts
- Display all saved contacts
- User-friendly graphical interface

---

## Technologies Used
- Java
- Swing
- AWT
- Object-Oriented Programming (OOPS)
- Data Structures

---

## Data Structures Used
- ArrayList
- LinkedList

These data structures are used for efficient storage and retrieval of contact information.

---

## Project Modules

### Add Contact
Allows users to save new contact details such as name and phone number.

### Search Contact
Enables fast searching of contacts using names or phone numbers.

### Edit Contact
Allows updating existing contact details.

### Delete Contact
Removes selected contacts from the phone book.

### Display Contacts
Shows all saved contacts in an organized format.
